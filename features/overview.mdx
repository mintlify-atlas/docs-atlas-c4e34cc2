## Highlight Features

### Auto-Restore for English

Type English words without switching off Vietnamese mode. Gõ Nhanh automatically detects and restores common English words.

```
text → text ✓   (not "têt")
user → user ✓   (not "úẻ")
push → push ✓   (not "púh")
```

<Note>
Press ESC after typing to manually restore any word: `user` → `úẻ` → **ESC** → `user`
</Note>

### Smart Per-App Mode

Gõ Nhanh remembers your ON/OFF preference for each application:

- Code in VS Code (OFF) → Switch to Slack (ON) → Back to VS Code (OFF)
- No manual toggling needed

### Automatic Input Source Following

Using multiple input sources? Gõ Nhanh automatically disables when you switch to:

- Japanese (あ)
- Chinese (中)
- Korean (한)
- Thai, Khmer, Lao, and other non-Latin scripts

Switching back to English automatically re-enables Gõ Nhanh.

## Compatibility

### Works Everywhere

Gõ Nhanh works in all macOS applications:

- **IDEs**: VS Code, Cursor, Zed, Windsurf, JetBrains
- **Browsers**: Chrome, Safari, Arc, Firefox
- **Productivity**: Notion, Obsidian, Bear
- **Communication**: Slack, Discord, Telegram
- **Terminals**: Terminal.app, iTerm2, Ghostty, Warp
- **AI Tools**: Claude Code, ChatGPT, Cursor AI

### Special Fixes

Built-in compatibility fixes for:

- Chrome/Arc address bar and search
- Spotlight search
- Claude Code in IDEs
- JetBrains IDEs

<Info>
For apps with input lag (like Electron apps), you can adjust the delay preset in **Settings → Advanced → Per-app config**.
</Info>

## Input Methods

<CardGroup cols={2}>
  <Card title="Telex" icon="t">
    Modern method using letter keys: `s`=sắc, `f`=huyền, `aa`=â, `w`=ư
  </Card>
  <Card title="VNI" icon="hashtag">
    Number-based method: `1`=sắc, `2`=huyền, `6`=â, `7`=ư
  </Card>
</CardGroup>

Both methods support:
- Flexible typing order (tone before or after modifier)
- Accurate tone placement following Vietnamese rules
- Auto-capitalization after `.` `!` `?` and Enter
