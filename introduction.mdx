  <p className="text-base dark:text-gray-400 text-gray-600 mb-8">
    Get up and running with Gõ Nhanh in minutes
  </p>
  
  <Steps>
    <Step title="Install Gõ Nhanh">
      Download and install via Homebrew or direct download
      
      ```bash
      brew install --cask gonhanh
      ```
    </Step>
    
    <Step title="Grant accessibility permissions">
      Open **System Settings → Privacy & Security → Accessibility** and enable Gõ Nhanh
      
      <Note>
        macOS requires accessibility permissions for keyboard input methods to function properly.
      </Note>
    </Step>
    
    <Step title="Start typing Vietnamese">
      The app automatically starts after installation. Look for the Gõ Nhanh icon in your menu bar and start typing!
      
      <Info>
        Type `text` and press space — it stays as `text`, not `têt`. Auto-restore handles common English words automatically.
      </Info>
    </Step>
  </Steps>
</div>
