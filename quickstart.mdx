Get up and running with GГө Nhanh in under 5 minutes. This guide covers installation, setup, and your first typing experience.

## Installation

<Steps>
  <Step title="Install via Homebrew (Recommended)">
    The easiest way to install GГө Nhanh is through Homebrew:

    ```bash
    brew install --cask gonhanh
    ```

    The app will automatically check for updates every 24 hours. You can also update manually:

    ```bash
    brew upgrade --cask gonhanh
    ```
  </Step>

  <Step title="Alternative: Manual Download">
    If you prefer not to use Homebrew, download the DMG file directly:

    1. Download [GoNhanh.dmg](https://github.com/khaphanspace/gonhanh.org/releases/latest/download/GoNhanh.dmg)
    2. Open the `.dmg` file
    3. Drag the app to your **Applications** folder

    <Warning>
      **macOS Security Warning**: If the app doesn't open, you need to remove the quarantine attribute. Run this command in Terminal:

      ```bash
      xattr -cr /Applications/GoNhanh.app
      ```

      This is a one-time step required for apps downloaded outside the App Store.
    </Warning>
  </Step>

  <Step title="Grant Accessibility Permissions">
    GГө Nhanh needs Accessibility permissions to intercept and transform your keystrokes.

    1. Open the app (it will launch automatically if installed via Homebrew)
    2. macOS will prompt you to grant permissions
    3. Click **Open System Settings**
    4. In **Privacy & Security вҶ’ Accessibility**, toggle on **GoNhanh**
    5. Restart the app

    <Info>
      The onboarding wizard will guide you through these steps automatically on first launch.
    </Info>
  </Step>
</Steps>

## First Use

### Choose Your Input Method

On first launch, you'll be prompted to choose between two input methods:

<CardGroup cols={2}>
  <Card title="Telex (Default)" icon="keyboard">
    The most popular Vietnamese typing method.

    - Vowels: `aa` вҶ’ Гў, `aw` вҶ’ Дғ, `ee` вҶ’ ГӘ, `oo` вҶ’ Гҙ, `ow` вҶ’ ЖЎ, `uw` вҶ’ Ж°
    - Tones: `as` вҶ’ ГЎ, `af` вҶ’ Г , `ar` вҶ’ бәЈ, `ax` вҶ’ ГЈ, `aj` вҶ’ бәЎ
    - Special: `dd` вҶ’ Д‘
  </Card>

  <Card title="VNI" icon="keyboard">
    Numeric-based input method.

    - Vowels: `a6` вҶ’ Гў, `a8` вҶ’ Дғ, `e6` вҶ’ ГӘ, `o6` вҶ’ Гҙ, `o7` вҶ’ ЖЎ, `u7` вҶ’ Ж°

<Note>

### Your First Typing Test

Once installed, try typing Vietnamese:

1. Open any text editor (Notes, TextEdit, browser, etc.)
2. Make sure GГө Nhanh is enabled (menu bar icon shows **VN**)
3. Type using Telex: `viet` `nam` вҶ’ "viб»Үt nam"
4. Type a full sentence: `xin` `chao` `ban` вҶ’ "xin chГ o bбәЎn"

## Switching Between Vietnamese and English

GГө Nhanh provides multiple ways to switch between Vietnamese and English:

### Toggle On/Off

Press **Ctrl + Space** (default) to toggle Vietnamese input on or off.

- **Enabled**: Menu bar shows **VN** (Vietnamese mode)
- **Disabled**: Menu bar shows **EN** (English mode)

<Note>
  You can customize this shortcut in **Settings вҶ’ General вҶ’ Shortcut**.
</Note>

### Auto-Restore Feature

GГө Nhanh automatically detects common English words and restores them when you press Space:

| You Type | Without Auto-Restore | With Auto-Restore |
|----------|---------------------|-------------------|
| `text `  | tГӘt                 | text вң“            |
| `expect ` | б»…pct               | expect вң“          |
| `with `  | Ж°ith                | with вң“            |
| `user `  | Гәбә»                  | user вң“            |
| `perfect ` | pб»Ғct              | perfect вң“         |

<Info>
  **How it works**: The engine analyzes each word after you type it. If the result contains invalid Vietnamese syllables, it automatically restores the original English input.
</Info>

### ESC Key Recovery

If auto-restore doesn't catch a word, press **ESC** to manually restore it:

1. Type `box` вҶ’ becomes `bГө` (valid Vietnamese)
2. Press **ESC** вҶ’ restores to `box`

### Smart Per-App Mode

GГө Nhanh remembers your language preference for each app:

- Coding in **VS Code** (Vietnamese OFF) вҶ’ typing stays in English
- Switch to **Slack** (Vietnamese ON) вҶ’ typing switches to Vietnamese
- Return to **VS Code** вҶ’ automatically switches back to English

<Note>
  Per-app mode is enabled by default. You can disable it in **Settings вҶ’ General вҶ’ Remember state per app**.
</Note>
